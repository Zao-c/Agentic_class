# ADR-0002：先建立无外部模型的本地检索基线

- 状态：已采纳
- 日期：2026-07-14

## 决策

首版使用中文字符、二元词和英文/数字标记的本地检索基线，并对题库来源降权、设备型号做元数据约束。文档和片段存入 SQLite，导入使用 SHA-256 去重。

## 原因

- 当前目录包含历史 FAISS 索引，但缺少可证明一致的模型和构建配置，直接复用不可复现。
- 基线无需下载模型，测试不会产生模型费用。
- 先固定数据、接口、评测和 Trace，后续 Embedding/精排实验才有可靠对照。

## 代价

这不是任务书目标态的 Embedding + BM25 + Cross-Encoder。语义改写、同义表达和复杂故障召回能力受限，必须在下一阶段用真实运行报告比较后再替换。
